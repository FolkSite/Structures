  6. Gallery
    A gallery can be any Chunk in your element tree; or a TV attached to this resource. - Just specify a correct title pattern (of ANY structure); - Element Abbreviation (not case sensitive) + element ID = Ch23 or TV34 - Use hyphen to optionally compound/suffix with other words; My Album - Ch23 My Gallery Etc - Tv34 - Then in a structure hover menu or main top nav, find Gallery (esc, g + 1)! Sample call for frontend display (using MIGX' getImageList) [[getImageList? &offset=`1` &where =`{"hidden:=":"0"}` &sort=`[{"sortby":"index","sortdir":"ASC","sortmode":"numeric"}]` &value=`[[$MyTuscanHoliday]]` // for raw data or chunk &tvname=`MyTuscanHoliday` // for TV when not using chunk &tpl=`MyTuscanHoliday_tpl` ]] *MyTuscanHoliday_tpl* `[[+folder]]` = url location of image - at this time you can use physical thumbnails: `[[+folder]]thumb200x100/` `[[+id]]` = image filename `[[+title]]` = image title extra info `[[+desc]]` = image description longer extra info `[[+tag]]` = image tag `[[+index]]` = image sort location img src="[[+folder]]thumb/[[+id]]" alt="[[+desc]]"
