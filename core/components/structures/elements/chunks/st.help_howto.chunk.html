Structures Content Area

  1. Blur structure to activate keyboard shortcuts; use esc key or click outside structures outermost dashed boundary lines
  2. Click or drag a structure title from the Structures Tree or use keys 1 through 9 - use alt to extend beyond 9
  3. Rearrange your structures by dragging its Index Number (on the left) or use up and down arrow keys
  4. Click the same Index Number to select/deselect a structure
  5. Double-click same Index Number to select/deselect all structures, or use a
  6. Click outside structures and drag across single or multiple structures to select them
  7. Click top of structure to enter a title (t) - or use the Drop Arrow button next to the Index Number
  8. To associate a Chunk (from element tree) or loaded resource TV, simply add - TV23 or CH23 to the title of your inserted structure, then run "Gallery" (g + 1) from your menu button

Drop Arrow Button

This button appears upon hover of each row, and thus signifies the active structure to which its menu is referring

If there is only one structure this button becomes absent, but its functions can still be keyboarded


Structures Main Menu / Save Buttons

Delete can also delete only selected structures. It defaults to permanently removing all structures from the face of the planet

Import: you will see your existing content in the source (s) textarea, or paste fresh content.
Make sure your content contains the official Structures MODX comment tags, that is, [[-STRUCTURES]] or [[-STRUCTURES-SINGLE]]
- Paste [[-STRUCTURES-SINGLE]] for only one block.
- Paste [[-STRUCTURES]] as many times where you want your content to be broken into blocks
When ready, hit Import (i + m)

NOTE! Structures uses the HTML in your st.importWrapper_tpl and st.importMarker_tpl chunks to decide what kind of editor to use

You may automatically accomplish this process of import/save resource by carefully selecting the import options in your Template's properties


Keyboard Shortcuts

  1. esc = Blur all Editors (or click outside) to activate Shortcuts
  2. ctrl + s = Save Resource
  3. ctrl + l.click = insert P in difficult areas of RTE
  4. b = Debug (shows structures detail in sidebar)
  5. h = Help / How-to Modal Window
  6. home = Toggle Distraction-free Mode
  7. up / down arrows = Move "this structure"
  8. c = Duplicate "this structure"
  9. del + 1 = Delete "this structure"
  10. del + a = Delete All/Selected structure(s)
  11. - / + = Shrink / Expand all structure(s)
  12. 0 = Total Count number of inserted structures
  13. 1 through 9 = Quick Insert of first NINE HTML template structures
  14. alt + single/double digits = Extends beyond first 9. Also, when a Rich Text Editor has focus, inserted structure becomes embedded
  15. s = Content Source and Import Manager
  16. i + m = To actually import/convert
  17. t = Enter title for "this structure" - add ch23 or tv23 to associate a gallery
  18. g + 1 = Activate gallery for "this structure"
  19. g + g = Activate Global Gallery (after which you can dblclick any Resource TV or any quick-updated Chunk from Element Tree
  20. g + t = Activate Default TV Gallery - of TV prespecified in Template Properties
  21. d = Disable Structures temporarily for this resource
  22. r + t = Quick Random Tip (per page load)
