Structures Content Area:

  1. Click or drag a structure title from the Structures Tree or use keys 1 through 9
  2. Blur structure so that keyboard shortcuts can work; use ESC key or click outside structures boundary line
  3. Rearrange your structures by dragging its Index Number (on the left) or use UP and DOWN arrow keys
  4. Click the same Index Number to select/deselect a structure
  5. Double-click same Index Number to select/deselect all structures, or use A
  6. Click outside structures and drag across single or multiple structures to select them
  7. Click top of structure to enter a title (or use the Drop Arrow button next to the Index Number)

Drop Arrow Button

This button appears upon hover of each row, and thus signifies the active structure to which its menu is referring


Misc / Clear / Save Buttons

Clear can also delete only selected structures. It defaults to permanently removing all structures from the face of the planet.


Keyboard Shortcuts

  1. ESC = Blur all Editors so that Keyboard shortcuts can work
  2. Ctrl + S = Save Resource
  3. H = Help Modal
  4. HOME = Toggle Distraction-free Mode
  5. UP / DOWN arrows = Move "this structure"
  6. C = Duplicate "this structure"
  7. DEL followed by 1 = Delete "this structure"
  8. DEL followed by A = Delete All structure(s)
  9. - / + = Shrink / Expand all structure(s)
  10. 1 through 9 = Quick Insert first NINE HTML template structures
  11. M = Misc : Open environment for Importing/converting
  12. I followed by M = To actually import/convert
